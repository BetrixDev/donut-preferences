# Donut Preferences

This is a project built with [Convex](https://convex.dev) as its backend.

## Project structure

The frontend code is in the `src` directory and is built with [React](https://react.dev/).

The backend code is in the `convex` directory.

`bun run dev` will start the frontend and backend servers.

`bun run setup.mjs` will setup the environment variables for the project.

`bun run build` will build the project for production.
